DiagnoSys

DiagnoSys provides a solution to diagnosing which drug is causing a user's adverse event. The user provides general information about themselves and the drugs their taking. DiagnoSys then compares thier query with thousands of FDA adverse events and outputs a rankedlist of the most-likely drugs to be causing the adverse event.

LaTex Tech

Using the power of character segmentation and convolusional neural networks, LaTex Tech accepts images of hand-writing or typed out equations and provides users with LaTex code that can be easily integrated within research papers, problem sets, or general notes.

Pigeonhole

Pigeonhole is a graphics based game similar to the children's game Battleship. Players alternate turns trying to uncover every hole on the map by throwing a bird at a gridspace. There are various birds worth various points, and players accumulate points by successfully uncovering a hole with a given bird.
